Development of "auction web application" using the Django and Vue (think of Ebay).

- Account creation and login working
- Using Django's AbstractUser model and authentication framework
- Using Vue router for frontend routing
- Modelling of application data, including users, items, bids, questions and answers
- Profile page included, with (editable) profile picture, email and date of birth	
- Page with a list of items currently available included and working as expected
- Users are able to send questions about items to the seller
- Using Ajax where required with Vue and fetch API
- Full use of static types both in Python and Vue
